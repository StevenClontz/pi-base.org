# About π-Base

The [π-Base model](https://lucant.org/papers/2025/241114-Clontz.pdf) facilitates
the curation of peer-reviewed and community-maintained knowledge in mathematics.

Underlying software is available on [GitHub](https://github.com/pi-base/).

## Projects using π-Base software

- [Topology.pi-Base.org](https://topology.pi-base.org)
- [pi-base-topos (topos theory)](https://github.com/pi-base-topos)

## Similar projects

- [DaRT: Database of Ring Theory](https://ringtheory.herokuapp.com/)
- [CatDat (category theory)](https://catdat.app/)

See also Terence Tao's [MathOverflow post listing crowd-sourced math research projects][0].
